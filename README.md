# ⏰ 피파 점검 대비 자동 관리기 (FC Online Auto Shutdown)

넥슨 FC 온라인의 목요일 정기 점검 때, 점검 시작 직전 넥슨 서버 시간에 맞춰 컴퓨터를 안전하게 종료하거나 다시 시작해 주는 프로그램입니다.

---

## ✨ 주요 기능
* **피파 서버 시간 동기화:** 네이비즘처럼 피파 공식 홈페이지 서버의 표준 시간을 읽어와 내 PC 시계와의 오차를 계산하므로 정확한 타이밍에 작동합니다.
* **작업 선택 기능:** 시간 도달 시 '시스템 종료' 또는 '다시 시작' 중 원하는 동작을 선택할 수 있습니다.
* **강제 종료 옵션(/f) 적용:** 피파 게임이나 다른 프로그램이 켜져 있어도 경고 창에 막히지 않고 무조건 강제로 안전하게 종료합니다.
* **직관적인 GUI:** 마우스 클릭과 숫자 입력만으로 누구나 쉽게 사용할 수 있습니다.

---

## 🚀 사용 방법
1. **`PC_Auto_Shutdown.exe`** 파일을 실행합니다.
2. 상단 메뉴에서 **[시스템 종료]** 또는 **[다시 시작]** 중 원하는 동작을 선택합니다.
3. 종료를 원하는 시간을 **24시간제** 형식으로 입력합니다. (예 : 오후 11시 30분 -> 23시 30분)
4. **[예약 시작]** 버튼을 누르면 서버 시간 동기화와 함께 예약이 완료됩니다.

> ⚠️ **주의 사항**
> 본 프로그램은 백그라운드에서 실시간으로 시간을 체크합니다. 창의 **'X' 버튼을 눌러 프로그램을 종료하면 예약도 함께 취소**되므로, 예약 후에는 창을 닫지 말고 **최소화(`-`)** 버튼을 눌러 내려두세요!

---

## 🛠️ 처음 실행 시 경고 창이 뜨는 경우 (필독)
본 프로그램은 개인이 개발한 실행 파일(`.exe`)로, 마이크로소프트 디지털 인증서가 등록되어 있지 않아 윈도우에서 안전하지 않은 파일로 오인할 수 있습니다. 

최초 실행 시 **"Windows의 PC 보호" (스마트스크린)** 파란색 창이 나타나면 아래와 같이 실행해 주세요. 바이러스나 악성코드가 아니니 안심하고 사용하셔도 됩니다.

1. 경고 창 본문 내부의 **[추가 정보]** 글자를 클릭합니다.
2. 우측 하단에 새롭게 나타나는 **[실행]** 버튼을 클릭합니다.
3. 이후부터는 경고 창 없이 더블클릭만으로 즉시 실행됩니다.
